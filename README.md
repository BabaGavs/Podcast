# Podcast

Welcome to the Podcast project!

## Overview
This project is designed to create and manage podcasts.

## Features
- User-friendly interface
- Multi-platform support
- Easy podcast management

## Installation
To install the Podcast project, run the following command:

```
git clone https://github.com/BabaGavs/Podcast.git
```

## Usage
After installation, navigate to the project directory and use the command:

```
cd Podcast
npm start
```
